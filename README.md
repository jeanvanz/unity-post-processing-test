# 🎨 Estudo de Post-Processing em Unity 2D

Este projeto foi desenvolvido com fins educacionais, com o objetivo principal de estudar e aplicar efeitos de **Post-Processing** para melhorar a qualidade visual e a atmosfera de uma cena 2D na Unity.

## 📌 Sobre o Projeto

O foco deste repositório não é em mecânicas complexas de jogo ou inteligência artificial, mas sim na composição visual. Utilizando assets gratuitos da comunidade, o projeto explora como a iluminação, correção de cores e efeitos de câmera podem transformar a "vibe" de um ambiente pixel art.

**O que foi explorado neste estudo:**
* Configuração de volumes globais de Post-Processing.
* Aplicação de efeitos como Bloom, Tonemapping, Shadows, Midtones entre outros.
* Organização da cena e composição de ambiente 2D.
* Movimentação básica do personagem para exploração do cenário.

## 🛠️ Tecnologias Utilizadas
* **Engine:** Unity (2D)
* **Pacotes:** Post Processing Stack
